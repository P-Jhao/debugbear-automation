# 项目说明（AGENTS）

## 功能边界

- 本项目是基于 Nuxt 的 DebugBear 批量性能测试工具。
- 核心能力包括：
  - 创建批量测试任务（URL、次数、版本、分组、设备等配置）。
  - 执行与调度任务，按运行结果汇总进度、成功/失败数量和指标。
  - 历史任务列表查询、筛选、详情查看、取消与删除。

## 核心目录结构

- `app/`：前端页面、组件、状态管理与请求封装。
- `server/api/`：服务端 API 路由。
- `server/utils/perfTask/`：任务配置、执行器、仓储与统计等核心逻辑。
- `shared/types/`：前后端共享类型定义。
- `.data/`：本地任务数据库与运行数据。
